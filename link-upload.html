🔗 Tautkan Sitasi ke Referensi

← Semua Tool

Unggah naskah .docx yang sudah berisi in-text citation dan daftar referensi. Setiap sitasi akan ditautkan (hyperlink internal) ke entri referensinya — klik sitasi di Word akan langsung meloncat ke referensinya. Teks & format artikel tidak diubah sama sekali, hanya sitasi yang jadi bisa diklik.

1Unggah Naskah (.docx)
📄
Klik untuk pilih file, atau seret & lepas di sini
.DOCX — DIPROSES SEPENUHNYA DI BROWSER ANDA, TIDAK DIUNGGAH KE SERVER MANA PUN
📄 ✕
2Gaya Sitasi
3Cakupan Link

Pilih bagian mana dari tiap sitasi yang ingin dibuat bisa diklik — tidak perlu highlight manual apa pun di Word, cukup pilih di sini.

4Warna Teks Sitasi (opsional)

Default: format asli tidak diubah sama sekali (cuma jadi bisa diklik). Kalau diaktifkan, teks sitasi yang ditautkan diberi warna — defaultnya biru khas hyperlink, bisa diganti bebas.

5Highlight Manual di Word (opsional, lanjutan)

Kalau Anda sendiri sudah men-highlight manual sebagian teks sitasi di file Word-nya (warna apa saja), sistem bisa mempersempit link ke bagian yang di-highlight itu SAJA — ini terpisah dari pilihan di atas, dan cuma berlaku kalau highlight-nya memang ada di file.

🔒 File Anda tidak pernah dikirim ke server mana pun — pembacaan & penulisan ulang .docx (via JSZip, langsung pada XML asli file) sepenuhnya berjalan di browser Anda. Heading daftar referensi harus tertulis persis "References" / "Daftar Pustaka" / "Bibliography" (dsb.) sebagai paragraf tersendiri sebelum daftar referensi dimulai.
