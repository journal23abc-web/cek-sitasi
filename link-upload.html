4Cakupan Tautan
🔒 File Anda tidak pernah dikirim ke server — semua pemrosesan (baca .docx, deteksi sitasi, penulisan hyperlink internal) berjalan langsung di browser Anda.

✅ Format & teks naskah tidak diubah sama sekali — sistem hanya menambahkan hyperlink internal (anchor) yang menyambungkan tiap in-text citation ke entri referensinya di daftar pustaka. Tampilan visual, layout, dan seluruh kata tetap identik dengan file asli.

ℹ️ Di Microsoft Word, hyperlink internal biasanya perlu Ctrl+Klik (atau Cmd+Klik di Mac) untuk melompat — ini perilaku standar Word untuk hyperlink di dalam teks berjalan, bukan keterbatasan alat ini.
